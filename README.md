### Running the Project
- Assuming you have Python (3.7.*) installed 
- create virutal environment and activate the environment full guide <a href="https://packaging.python.org/guides/installing-using-pip-and-virtual-environments/">here</a>

#### **clone the repository**:<br>
`git clone https://github.com/krunal-ctrl/Rajkot_Hackathon`<br>

#### **install requirments**: <br>
`pip install -r requirements.txt`

#### **run rasa on GUI mode**: <br>
`rasa x`
